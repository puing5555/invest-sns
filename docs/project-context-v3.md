## ⭐ 내 종목 대시보드 (Part 1, 핵심)

### 메인 화면: 실시간 타임라인
```
관심종목 칩: [전체] [삼성전자 +0.8%] [현대차 +2.1%] [SK하이닉스 -1.3%] ...

🔴 실시간 업데이트 (내 관심종목에 대한 모든 이벤트)
  삼성전자 [A등급] 자사주 5조원 취득 결정 — 3분 전
  현대차 [🔵매수] 슈카월드 — 15분 전
  SK하이닉스 [리포트] 미래에셋 — 42분 전
  테슬라 [서프라이즈+18%] 4Q 매출... — 1시간 전
  ...
```

### 타임라인 아이템 클릭 시: 해당 종목 + 해당 탭으로 이동
```
삼성전자 A등급 공시 클릭 → 삼성전자 > 공시 탭
현대차 🔵매수 슈카월드 클릭 → 현대차 > 인플루언서 탭
SK하이닉스 리포트 클릭 → SK하이닉스 > 리포트 탭
테슬라 서프라이즈 클릭 → 테슬라 > 실적 탭
삼성전자 임원매수 클릭 → 삼성전자 > 임원매매 탭
```

### 종목 상세 대시보드 (탭 구조, 확정)
```
삼성전자  005930.KS · KOSPI                    ₩71,200 +1.14%

[실시간] [인플루언서] [애널리스트] [공시] [실적] [리포트] [임원매매] [일정] [메모]
```

**실시간 탭** = 해당 종목의 모든 업데이트 타임라인 (공시+신호+리포트+실적 다 섞임)
**인플루언서 탭** = 차트 + 신호 마킹 + 발언 테이블
**애널리스트 탭** = 차트 + 목표가 마킹 + 리포트 테이블
**나머지 탭** = 각 카테고리별 상세

### 인플루언서 탭 상세 (확정)
```
기간 필터: [1개월] [6개월] [1년] [3년] [전체]
인플루언서 필터: [전체] [슈카월드(8)] [김투자(5)] [박주식(3)] [이효석(12)]

[차트 + 신호 점 마킹]
[신호 범례: 🔵매수 🟢긍정 🟡중립 🟠경계 🔴매도]

인플루언서 필터 클릭 시:
  - 전체: 모든 인플루언서 신호 + 전체 테이블 (인플루언서 컬럼 포함)
  - 슈카월드(8): 슈카 신호만 차트에 + 슈카×해당종목 테이블만
    → 테이블 컬럼: 신호 | 분석 | 날짜 | 수익률 | 영상
    → "슈카월드 프로필 보기 →" 링크
```

```
  - 신호 대신 투자의견 (매수/중립/매도) + 목표가

```

## 👤 인플루언서 신호 시스템 (상세 설계 확정)

### 신호 체계 (5개, 확정)
```
🔵 매수    "사라, 담아라, 들어가라" (명확한 액션)
🟢 긍정    "좋아보인다, 괜찮다, 관심가져라"
🟡 중립    "지켜보자, 모르겠다, 반반이다"
🟠 경계    "조심해라, 리스크 있다, 주의"
🔴 매도    "팔아라, 손절해라, 빠져라" (명확한 액션)
```

### 핵심 규칙
- **영상 1개 × 종목 1개 = 신호 1개** (절대 규칙)
- 한 유튜버가 같은 종목을 여러 영상에서 언급 → 각각 별개 신호 (의견 변화 추적)
- 수익률 = 발언 시점 주가 → 현재 주가 (팩트만 표시, 적중률 아님)

### 전체 파이프라인 (8단계, 확정)
```
STEP 1: 영상 감지 (30분마다 YouTube API)
STEP 2: 자막 추출 (YouTube 자동자막 → 없으면 Whisper)
STEP 3: 전체 영상 요약 (문맥 보존, 종목 자르기 전에 먼저)
STEP 4: 종목 추출 + 신호 동시 판단 (요약 + 원본 자막 입력)
STEP 5: AI 반박 검증 (1차 분류 → 2차 반박 → 불일치 시 3차 타이브레이커)
STEP 6: DB 저장 + 차트 마킹
STEP 7: 유저 피드백 수집 (신고 시스템)
STEP 8: 주간 프롬프트 자동 고도화
```

### 영상 삭제 시
- "영상 삭제됨" 표시
- 자막 기록 + AI 분석 결과는 DB에 보관
- 신호 점도 차트에 유지
- 유저에게는 AI 요약 + 핵심 발언(1-2문장)만 보여줌 (전체 자막 원문은 내부 DB만)

### 라이브 방송
- 라이브 중 실시간 분석 ❌
- VOD 전환 후 자막 있으면 일반 영상처럼 처리

### 과거 영상 수집 (런칭 전)
```
채널에 있는 모든 영상 다 긁어옴
처리 순서: 최근 → 과거 역순, 인기 유튜버부터
자막 있는 영상 먼저 (1주), 없는 영상 Whisper (2주)
유튜버 선정은 창업자가 직접 리스트업
```

### AI 교차 검증 (반박 방식, 확정)
```
1차: 신호 분류 → 🟢긍정
2차: 반박 프롬프트 "1차 결과를 검증해라. 틀릴 수 있는 이유를 찾아라."
  → 일치(90%) → 확정
  → 불일치(10%) → 3차 타이브레이커
```

### 면책 + 내부 검토
```
면책: "AI 자동 분류, 부정확할 수 있음. 원문 확인 필수."
내부 검토: 초기 2주 창업자가 매일 30분, 이후 주간 30분 (신고된 것만)
```

```
  ❌ 인플루언서 발언 추적/차트 마킹
  ❌ 컨센서스 대비 서프라이즈 분석
  ❌ 증권사 리포트 AI 요약
  ❌ 관심종목 기준 통합 대시보드

토스가 유튜버 추적을 못 하는 이유:
  1) 인플루언서 앰버서더 프로그램과 충돌
  2) 유튜버 항의 시 브랜드 리스크
  3) AI 오분류 시 소송 리스크
  4) 매매 수수료와 직접적 연관 없음
```
