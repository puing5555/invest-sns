## 추출 방식 설명

- **전체추출**: 투자 키워드 필터만 적용 (종목명 무관하게 투자 관련 영상 전체 분석)
- **부분추출(stock-filter)**: 종목명이 제목에 포함된 영상만 분석 (매크로/교육 비중 높은 채널에 적합)

## dry-run 보고 규칙 (Rule 18)

새 채널 dry-run 시 반드시 3가지 수치 보고:
1. 투자 키워드 필터 통과 건수
2. --stock-filter 통과 건수 (종목명 포함)
3. 비율 (stock-filter / 키워드필터)

| 비율 | 권장 |
|------|------|
| 20% 이하 | --stock-filter 권장 |
| 50% 이상 | --execute 권장 |
| 20~50% | JAY 판단 |
